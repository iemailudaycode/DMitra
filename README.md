# D-Mitra : Defense Recruitment Portal

# Project Overview
D-Mitra is a web-based recruitment portal for defense candidates.  
Tech Stack: PHP, MySQL, HTML, CSS, JavaScript  

Key Features:
- Candidate registration and login
- Admin dashboard with candidate management
- Eligibility filtering and reporting
- Secure and structured database

# How to Run Locally
1. Clone the repository
2. Import `db.sql` into your local MySQL server
3. Configure database connection in `db.php`
4. Run the project on XAMPP / WAMP
